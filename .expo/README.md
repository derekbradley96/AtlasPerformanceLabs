# This project is Vite + Capacitor, not Expo

Atlas Performance Labs uses **Vite** for the build and **Capacitor** for native iOS/Android. It does **not** use Expo.

- **Web:** `npm run dev` (Vite dev server)
- **iOS:** `npm run ios` or `npm run open:ios`
- **Android:** `npm run android`
- **Live reload (iOS):** `npm run ios:live` (Capacitor loads from localhost)

The `.expo` folder may exist from tooling; ignore it. Do not use `expo start` or Expo-specific commands.
